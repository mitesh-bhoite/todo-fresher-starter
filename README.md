# Todo App

This is a simple Todo List application made with **HTML, CSS, and JavaScript**.  
It allows you to create tasks and add **Subtasks** under each main task.

The UI is clean and built in **black and white theme**.

## > Features

    - Add main tasks
    - Add **subtasks**
    - Mark tasks as completed
    - Delete tasks and subtasks
    - Filter tasks:
        - All
        - Active (Not completed)
        - Completed
    - Drag and drop tasks:
        -  Move parent tasks
    - Save tasks in **LocalStorage**
    - Fully responsive UI (Desktop + Mobile)

    --

    ## Tech stack

    |    Part    | Technology                       |
    |------------|----------------------------------|
    | Frontend   | HTML, CSS, JavaScript            |
    | Storage    | 'localStorage' (Browser storage) |
    | Deployment | Netlify / GitHub Pages           |
